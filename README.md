# 바닐라 JS 미니 프로젝트 모음

빌드 도구나 프레임워크 없이 순수 HTML / CSS / JavaScript만으로 만든 미니 프로젝트 모음입니다.

## 프로젝트 목록

### ✅ 할 일 목록 (`to-do/`)
할 일을 추가·완료·삭제할 수 있는 Todo 앱입니다.
- 항목 추가 / 완료 체크 / 삭제
- 전체 · 진행중 · 완료 필터링
- 전체/완료 개수 통계
- 완료된 항목 일괄 삭제

### 🔢 카운터 (`counter/`)
버튼으로 숫자를 올리고 내리는 카운터 앱입니다.
- 증가 / 감소 / 리셋 버튼

### 💱 환율 변환기 (`fxconverter/`)
USD, EUR, KRW, JPY 간 환율을 계산해 주는 앱입니다.
- 금액 입력 후 통화 선택
- 통화 교환(swap) 버튼
- 변환 결과 즉시 표시

### 🐍 스네이크 게임 (`snake/`)
캔버스 기반의 클래식 스네이크 게임입니다.
- 방향키로 뱀 조종
- 쉬움 / 보통 / 어려움 / 매우 어려움 난이도 선택
- 점수 · 레벨 · 최고 점수 표시
- P 키 또는 버튼으로 일시정지

### 🌐 헬로 월드 (`hello-world/`)
파란 그라데이션 배경의 미니멀 스타터 페이지입니다.

### 👤 프로필 (`profile/`)
보라색 테마의 개인 프로필 카드 페이지입니다.

## 실행 방법

각 폴더의 `index.html` 파일을 브라우저에서 바로 열면 됩니다. 별도의 서버나 설치가 필요 없습니다.

```
to-do/index.html
counter/index.html
fxconverter/index.html
snake/index.html
hello-world/index.html
profile/index.html
```

## 기술 스택

- HTML5
- CSS3 (CSS 변수, Flexbox)
- JavaScript (ES5 스타일, 바닐라)
